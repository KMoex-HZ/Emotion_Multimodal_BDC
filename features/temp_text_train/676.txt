 Cacar air rencajar api sama sama disembakan oleh infeksi virus herpes, ya itu parisi alajoster. Saat pertama kali Anda terinfeksi virus parisi alajoster, Anda akan mengalami cacar air. Sementara itu, bila Anda terjangkit virus ini aktif kembali untuk yang kedua kali atau lebih, pangka Anda akan mengalami cacar api. Orang yang terkena cacar api, biasanya mengalami satu atau lebih dari hijala berikut. Pulit melepuh berisi cairan, rasasakit seperti terbakar dan menusuk, rasak kesemutan, gata atau mati rasa apada kulit, mengigil, demam, sakit kepalah atau sakit perut. Ada apa 3 paktor esiko untuk mengalami cacar api? Ya itu, busia di atas 50 tahun, sistem imun yang melemah dan memiliki kondisi medis yang mendurunkan sistem imun. Herpes, zoster atau cacar api juga berisi ko menyebapkan komplikasi serius. Seperti nyarisar api jaga panjang dan kehilangan penglihatan, pasien diarbetas meniliki risiko 20% lebih besar, mengalami cacar api. Oleh karena itu, segera risiko cacar api dengan menjaga imun tubuh dengan mengurangi stress, pengkonsumsi makanan bergizi, menjaga polah hidup sehat dan melakukan faksinasi. Segera berkonsultasi dengan tenaga ke sehatan profesional untuk mengetahui penanganan cacar api yang tepat.
