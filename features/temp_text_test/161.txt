 Sobat Sehat, Tahukah Anda, orang yang berusia 5,000 tahun ke atas berisi kotinggi mengalami cacar api. Penyakit dengan nama lain Herpes Zoster atau cacar api ini, menijur ruang kulit yang menyakitkan. Lalu, apa penyebab cacar api dan bagimana cara motohi bijalannya? Cacar api disebabkan virus yang sama seperti cacar air di atu virus Farisela Zoster. Cacar api terjadi ketika virus Farisela Zoster dalam tubuh aktif kembali, setelah sebelumnya sesu orang pernah terkena cacar air. Gejala cacar api berfariasi pada setiap orang, baik ringan maupun berat. Orang yang terkena cacar api biasanya mengalami satu atau lebih dari dijala berikut. Kulit melepuh berisi cairan, rasa sakit seperti terbakar dan menusup, rasa kesemutan, gatal atau mati rasa pada kulit, mengigil, demam, sakit kepalah atau sakit itu. Menurut data dari CDC, satu dari tiga orang, menderita cacar api setidaknya satu kali seumur hidupnya. Herpes Zoster atau cacar api juga berisiko menyabakkan komplikasi serius, seperti nyarisarap yang kapan yang dan kehilangan penglihatan. Penelitiang terditan human vaccines dan immunotera petik menunjukkan, pasien diabetes memiliki risiko 20% lebih besar mengalami cacar api. Pasalnya diabetes menyabakkan sel-sel yang berperan dalam respons imun mengalami gangguan fungsi, sehingga meningkatkan risiko mengalami cacar api. Tidak hanya itu cacar api juga memperburup dengan dalihan buladara bagi pasien diabetes. Penuh diketahui juga, risiko cacar api meningkat seiring bertambahnya usia dan pada orang yang memiliki sistem kekebalan tubuh yang nge-mah. Oleh karena itu, cega risiko cacar api dengan menjaga imun tubuh dengan mengorangi stress, fungsi makanan bergizi, menjaga polah hidup sehat dan melakukan vaccinasi. Segera berkonsultasi dengan tenaga kesehatan profesional untuk mengatahui penanganan cacar api yang tepat.
